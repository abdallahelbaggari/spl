SPL

Privacy Policy · Last updated: June 2026

1. Introduction

SPL ("the App") is a global football super app built for the Pi Network ecosystem. This Privacy Policy explains how we collect, use and protect your information when you use SPL.

2. Information We Collect

  • Pi Network username (via Pi SDK authentication)
  • App usage data (selected competitions, predictions, preferences)
  • Transaction records (Pi payment references only)

3. Pi Network Integration

SPL uses Pi SDK exclusively for authentication and payments. We do not collect passwords, private keys, or wallet data. All Pi transactions are processed through Pi Network infrastructure.

4. Data Storage

User preferences and prediction scores are stored locally on your device using browser localStorage. We do not operate external user databases.

5. Third-Party Services

SPL uses the following services to provide live football data:

  • football-data.org — match data and standings
  • ESPN (unofficial API) — live scores and news
  • The Guardian API — football news
  • Wikimedia Commons — football images (CC Licensed)

6. Pi Ecosystem Compatibility

SPL uses Pi SDK exclusively for all authentication and transactions. The app does not support any other login methods or payment currencies.

7. No Gambling or Betting

SPL's prediction feature is for entertainment only. No real money, cryptocurrency or Pi is wagered. Predictions earn internal points only.

8. Contact

For privacy questions: support.spl.pi@gmail.com
